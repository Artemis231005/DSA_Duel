# ⚔️ DSA Duel

> A competitive DSA practice tracker where two friends battle their way to better coding skills through streaks, leaderboards, achievements, analytics, and AI-powered coaching.

---

## 📖 Overview

**DSA Duel** transforms Data Structures & Algorithms practice into an engaging competition.

Instead of simply solving problems, users compete in real time by earning points, maintaining streaks, unlocking achievements, tracking analytics, and receiving personalized AI coaching.

Whether you're preparing for coding interviews or improving problem-solving consistency, DSA Duel keeps you motivated through friendly competition.

---

## ✨ Features

### 🏆 Competitive Leaderboard
- Live head-to-head rankings
- Dynamic point system
- Weekly winners
- Daily streak tracking
- Streak bonus rewards

### 📝 Question Logger
- Supports multiple coding platforms
  - LeetCode
  - HackerRank
  - GeeksforGeeks
  - CodeStudio
- Difficulty tracking
- Topic categorization
- Revision mode
- Weekly practice goals

### 📅 Activity Tracking
- GitHub-style contribution heatmap
- Interactive monthly calendar
- Daily submission history
- Practice consistency visualization

### 📊 Analytics Dashboard
- Total solved questions
- Points progression charts
- Difficulty distribution
- Topic-wise comparison
- Performance trends
- Mastery indicators

### 🏅 Achievement System
Unlock badges by completing milestones such as:

- 3-Day Streak
- 7-Day Streak
- Array Master
- Graph Guru
- Dynamic Programming Expert
- Legendary Achievements

Achievements are categorized into:

- 🟤 Bronze
- ⚪ Silver
- 🟡 Gold
- 💎 Platinum
- 🔷 Diamond
- 👑 Master

### 🤖 AI Performance Coach
Powered by **Gemini 3.5 Flash**, the AI coach provides:

- Personalized performance analysis
- Goal completion prediction
- Burnout detection
- Weak topic identification
- Practice recommendations
- Interactive coaching chatbot

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/dsa-duel.git
```

Go inside the project

```bash
cd dsa-duel
```

Install dependencies

```bash
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file.

```env
GEMINI_API_KEY=your_api_key
```

If no API key is provided, the application automatically switches to offline fallback mode.

---

# ▶️ Run Locally

```bash
npm run dev
```

Open

```
http://localhost:3000
```

---

# 📦 Production Build

Build

```bash
npm run build
```

Start

```bash
npm start
```
